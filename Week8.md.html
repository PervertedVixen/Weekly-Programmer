Weekly Programmer - Week 8 ========================== CryptoQuips ----------- From [Weekly Programmer Challenges](./index.html) Description ----------- Write a program that produces a single letter ciphertext, sometimes known as a cryptogram or a cryptoquip, for a given input. Formal Inputs & Outputs ----------------------- ### Input Description Input will be a string which may be read in the manner that best suits your program. ### Output description Output will be the cyphertext, which may include a hint to the solution. Example Inputs/Outputs ---------------------- ```text > the quick brown fox jumped over the lazy dog qco htvsb fipzg jpe rtadok pwoi qco yunx kpm o=e ```
