# Weekly Programmer - Week 13 ## Polygons ## Challenge Description Given a list of points in 2 dimensions determine whether the list represents a valid polygon. For the purposes of this program a valid polygon is one where no edge intersects another. For Example the input `0,0 1,0 0,1 1,1` does not represent a valid polygon because `1,0 0,1` and `0,0 1,1` intersect. ## Bonus Challenge 1) When provided with an invalid polygon, alter the order of the input points such that the result *is* a valid polygon. 2) Find the area of the valid polygon ## Challenge Inputs 1) `0,0 1,0 0,1 1,1` 2) `0,0 1,0 1,1 0,1` 3) `0,0 1,0 0.50000001,0.5 1,1 0,1 0.49999999,0.5` 4) `0,0 1,0 0.49999999,0.5 1,1 0,1 0.50000001,0.5` 5) `0,0 4,2 2,6 1.5,4 0.5,4 -4,4 -3,4 -3,2` 6) `0,-4 -3,0 3,3 4,-1 -2,4 -2,-3 5,1 1,4`
